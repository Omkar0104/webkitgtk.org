WebKitSoupAuthDialog

WebKitSoupAuthDialog — A SoupFeature to provide a simple authentication dialog for HTTP basic auth support.

Synopsis

                    WebKitSoupAuthDialog;

Description

WebKitSoupAuthDialog is a SoupFeature that you can attach to your SoupSession to provide a simple authentication dialog, with optional GNOME Keyring support, while handling HTTP basic auth. It is built as a simple C-only module to ease reuse.
