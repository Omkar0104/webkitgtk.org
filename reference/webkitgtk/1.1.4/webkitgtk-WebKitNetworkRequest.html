WebKitNetworkRequest

WebKitNetworkRequest — The target of a navigation request

Synopsis

                    WebKitNetworkRequest;
const gchar *       webkit_network_request_get_uri      (WebKitNetworkRequest *request);

Description

This class represents the network related aspects of a navigation request. Currently this is only the uri of the target. In the future the state of the web form might be added. Currently this object is only used along with the "navigation-requested" signal.
